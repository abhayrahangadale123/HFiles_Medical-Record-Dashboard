# Mini Medical Record Dashboard
**Assignment for HFiles — Developer Candidate: Abhay Rahangadale**

A simplified Medical Record Dashboard (Frontend: Next.js, Backend: ASP.NET Core).  
This repository is submitted as part of the HFiles assignment. The project demonstrates user signup/login (session cookie), profile management, file upload (medical records), and a dashboard to list/search/view/delete records.

- frontend — **Frontend** (Next.js) code. Folder: `frontend/`
